Bulk Upload

How Bulk Upload Works

  • Upload a CSV file with interviewee emails and project assignments
  • New projects are automatically created for each unique topic
  • Interviews can also be added to existing projects using their ID
  • Invalid rows are skipped (check server logs for details)

Upload a CSV file with project and interviewee information.

Optional. Interview type to use for newly created projects.

CSV Column Reference
Column Description
emailrequired Interviewee's email address
nameoptional Interviewee's display name (uses email prefix if not provided)
project_topicoptional Creates a new project with this topic (or reuses if already exists in this import)
project_idoptional UUID of an existing project to add the interview to

Each row must have either project_topic or project_id to assign the interview.

Example CSV
email,name,project_topic
alice@example.com,Alice Smith,Customer Feedback Research
bob@example.com,Bob Johnson,Customer Feedback Research
carol@example.com,Carol Williams,Product Usability Study
dave@example.com,,Product Usability Study

This example creates 2 projects: "Customer Feedback Research" (with Alice and Bob) and "Product Usability Study" (with Carol and Dave).
