Bulk Import

How Bulk Import Works

  • Upload a CSV file with guest emails and interview assignments
  • New interviews are automatically created for each unique topic
  • Guests can also be added to existing interviews using their ID
  • Invalid rows are skipped (check server logs for details)

Upload a CSV file with interview and guest information.

Optional. Template to use for newly created interviews.

CSV Column Reference:

Column Description
email required Guest's email address
name optional Guest's display name (uses email prefix if not provided)
interview_topic optional Creates a new interview with this topic (or reuses if already exists in this import)
interview_id optional UUID of an existing interview to add the guest to

Each row must have either interview_topic or interview_id to assign the guest.

Example CSV:

email,name,interview_topic
alice@example.com,Alice Smith,Customer Feedback Research
bob@example.com,Bob Johnson,Customer Feedback Research
carol@example.com,Carol Williams,Product Usability Study
dave@example.com,,Product Usability Study

This example creates 2 interviews: "Customer Feedback Research" (with Alice and Bob) and "Product Usability Study" (with Carol and Dave).
